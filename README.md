# Quy trình Clone & Hoàn thiện App (VN_NAMI)

Guide 7 bước clone + hoàn thiện app Android trên scaffold VN_NAMI:
cài app mẫu → clone → hoàn thiện tính năng → đặt ads → IAP → language → test bằng AI agent.

Có form nhập thông số dự án; mọi bước tự cập nhật theo giá trị nhập.

👉 Xem bản web: mở `index.html` hoặc truy cập GitHub Pages của repo.

> Bản public đã ẩn AdMob App ID và path máy; điền giá trị thật của bạn khi dùng.
